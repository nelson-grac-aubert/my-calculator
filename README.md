# Python Terminal Calculator

## Description
Calculatrice qui permet à l'utilisateur d'écrire des expressions mathématiques et de les évaluer

## Prérequis
Python

## Fonctionnalités
- Effectuer un calcul (+, -, *, /)
- Consulter l'historique
- Effacer l'historique
- Quitter le programme
